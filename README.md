# Project_1_Data_Cleaning
Data cleaning project on Lagos Healthcare dataset

Project Overview

This projects focuses on cleaning a raw dataset related to healthcare services in Lagos, Nigeria. The objective was to ensure data integrity and make sure the data is perfectly cleaned.

Tools Used:Python (Pandas)

Key Steps Taken

Data Inspection: Identified missing values, duplicate entries and inconsistent formatting.

Data Cleaning: Handled null values, corrected data types and standadized naming conventions to ensure consistency.

Validation: Performed final checks to ensure the dataset is accurate and reliable for downstream analysis.

Files Included

Project_1_Data_Cleaning.ipynb: The complete Python code and analysis process.

Project_1_Data_Cleaning.pdf: A summary report of the cleaning findings.

Project_1_Report_log.pdf: Detailed documentation of the cleaning log.
